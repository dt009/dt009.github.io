html 标签嵌套

发表于 2018-02-07 | 分类于 HTML 知识 / 基础

[分享] html 标签嵌套

起因

p 标签内不能使用 div 标签

HTML（超文本标记语言）中元素大多数都是“块级”元素或行内元素。

块级元素

块级元素占据其父元素（容器）的整个空间，因此创建了一个“块”

从上往下排列

块级元素前后另起一个新行

可以设宽高

padding/margin 都起作用

内联元素

一个行内元素只占据它对应标签的边框所包含的空间;

再同一行展示, 新的行内元素不会新启一行

不可以设置宽和高, 大小为内容撑起的大小

padding/margin 的上下值不起作用

置换元素

img / input

一般情况下:

行内元素只能包含数据文本和其他行内元素

而块级元素可以包含行内元素和其他块级元素

其他情况

一些块元素不能包含一些块元素

把块元素分为三个级别

第一级别: body

第二级别: 可以自由嵌套

第三级别: 不可以自由嵌套 ( 元素的内容只能放内联元素和特定的元素 )

标题标记: caption, h1, h2, h3….

段落标记: p

特定元素: ul(li), ol(li), dl(dt | dd), table(caption | thead | tbody | tfoot), tr(只存在于 thead | tbody | tfoot 中之后才是 td | th)
