注意: 在安装过程中又一次输入 mac 用户密码的地方

安装成功后可以使用 brew 命令了

简单的命令

  1. 安装软件：brew install 软件名;
  2. 搜索软件：brew search 软件名;
  3. 卸载软件：brew uninstall 软件名;

  4. 更新所有软件：brew update;

    update 可以把包信息更新到最新，不过包更新是通过git命令，所以要先通过 brew install git 命令安装git。

  5. 更新具体软件：brew upgrade 软件名;
  6. 显示已安装软件：brew list;
  7. 查看软件信息：brew info／home 软件名;
  8. 查看那些已安装的程序需要更新： brew outdated;
  9. 显示包依赖：brew reps;
