css盒模型(标准和怪异盒模型)

发表于 2018-01-25 | 分类于 css 基础

盒模型

组成

  1. content: 内容区块;
  2. padding: 内边距;
  3. border: 边框;
  4. margin: 外边距;

标准盒模型和怪异盒模型

盒模型分为两种:

  1. W3C 标准盒模型;
  2. IE 标准盒模型;

怪异模式

部分浏览器在支持W3C标准的同时还保留了原来的解析模式;

主要表现在 IE 内核的浏览器, 当不对 Doctype 进行定义的时,触发 怪异模式.

计算

标准盒模型

> 块的总宽 = width + margin(左右) + padding(左右) + border(左右);

怪异和模型

> width = content的宽 + padding(左右) + border(左右);

> 块的总宽 = width + margin(左右);

box-sizing 属性

  • 语法

  • 当设置 border-box 时, 采用怪异模式解析计算;
  • 当设置 content-box 时, 采用 w3c 标准解析计算;

    建议初始化是进行 box-sizing: border-box; 设置.
